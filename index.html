El Tapiz de Mi Vida - Laura Farfán

Una experiencia web para presentar una historia personal de forma visual, emocional y organizada por capitulos.

Resultado

El sitio transforma una historia personal en una experiencia digital profesional, memorable y accesible desde cualquier dispositivo.

HTML, CSS, JavaScript, responsive design, GitHub Pages.
